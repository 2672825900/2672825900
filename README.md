### Features
- [x] Search a torrent. 
- [x] Stream videos and view images.
- [x] Filter Torrents
- [x] get files by magnet link
- [ ] Download files 
- [x] Chromecast support
- [ ] View files like pdf, text etc.

### Development

To start the development server run `yarn start`
